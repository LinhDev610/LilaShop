## Description

E-commerce platform for cosmetics retail with order management system.

---

## Responsibilities

Design, build, and maintain REST APIs for the React web app.

Implement authentication: login/logout, registration, forgot/reset password with email OTP.

Develop CRUD services for accounts/users, categories, and products with variants and inventory management.

Deliver product/catalog workflows: approval, restock, search/filter, and media uploads.

Integrate payments/shipping and incentives: MoMo payment gateway, GHN shipping API, vouchers/promotions.

Handle orders/returns: shopping cart, multi-step checkout, order lifecycle, returns/refunds, and notifications.

Build customer support systems: ticket management, complaint handling, and review moderation.

Provide reporting endpoints: revenue statistics, financial reports, and business analytics.

Develop React frontend: responsive SPA with role-based admin panel, staff panel, and customer portal.

Set up platform foundation: Spring Boot layered architecture, security profiles, and Docker containerization.

---

## Technologies

**Backend:** Java, Spring Boot, Spring Data JPA, Spring Security + JWT

**Frontend:** React

**Database:** MySQL

**Build/Tools:** Maven, Docker

**Integrations:** MoMo payments, GHN shipping, email/OTP
